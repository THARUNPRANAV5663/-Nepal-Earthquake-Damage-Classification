# -Nepal-Earthquake-Damage-Classification
Built a multiclass classification model to predict structural damage grades of buildings from the 2015 Nepal Gorkha Earthquake using LightGBM. Applied EDA, outlier capping, SMOTE for class imbalance, Optuna for hyperparameter tuning, and custom threshold tuning — achieving a Macro F1 score of 0.7056 on ~260K building records.
